Unity with MRTK3 v3.1 Help

Setup d'un projet Unity

Créer un projet

La création d'un projet Unity a son importance, car c'est sur celle-ci que le projet va reposer. Une erreur peut avoir une conséquence casquade et la correction de ce type d'erreur est complexe.

Création du projet

  1. Cliquer sur "New project"

  2. Entrer le nom du projet, sa location sur votre PC et décocher "Unity Cloud"

  3. Vous pouvez maintenant créer votre projet

Setup du build

Lorsque votre projet a fini de charger, vous arrivez sur l'environnement Unity.

Avant de commencer le projet, il faut modifier les settings pour avoir un environnement de travail adapté à ses besoins.

  1. Cliquer sur File -> Build Settings

  2. Sélectionner Universal Windows Platform puis cliquer sur Switch Platform

  3. Configurer le UWP de la manière suivante :

    Vous renseignerez l'IP de votre casque Hololens 2 ainsi que vos identifiants dans cette section.

    • Trouver l'IP de son Hololens 2

    • Créer un username et un password

    La fenêtre peut maintenant être fermée

  4. Cliquer sur Edit -> Project Settings -> XR Plugin Management puis cliquer sur installer

  5. Dans le Project Settings, cliquer sur Player puis ajouter si demandé
